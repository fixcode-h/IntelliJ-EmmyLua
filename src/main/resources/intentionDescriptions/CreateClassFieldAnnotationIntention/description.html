Creates class and field annotations for local variables that are returned at the end of the file.

This intention action analyzes the current file to find all self. field references and generates appropriate @class and @field annotations for the local variable.

Example: 

local BP_RebornInteractVolume_C = {}
function BP_RebornInteractVolume_C:new()
    self.Age = 0
    self.Name = "test"
end
return BP_RebornInteractVolume_C
Will be transformed to: 
---@class BP_RebornInteractVolume_C
---@field Age number
---@field Name string
local BP_RebornInteractVolume_C = {}
function BP_RebornInteractVolume_C:new()
    self.Age = 0
    self.Name = "test"
end
return BP_RebornInteractVolume_C
